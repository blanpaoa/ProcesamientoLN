## Cómo ejecutar

1. Clonar el repositorio:
   ```bash

   cd nlp
   ```

2. Instalar dependencias:
   ```bash
   pip install numpy scikit-learn jupyter
   ```

3. Abrir el notebook deseado:
   ```bash
   jupyter notebook desafio_1/Desafio_1_LenguajeNatural.ipynb
   ```
